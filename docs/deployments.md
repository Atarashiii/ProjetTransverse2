# Déploiement

## Processus de déploiement

1. Pull du code depuis le dépôt GitHub.
2. Exécution des tests unitaires et de linting.
3. Build de l'application.
4. Déploiement en environnement de staging.
5. Exécution des tests en staging.
6. Déploiement en environnement de production.
7. Notifications en cas de succès ou d'échec.
